# Mall-system-Management-terminal-user-terminal-python-react
# python-电商商城平台项目源码（管理端+PC用户端）
### 1、介绍
本项目是基于现代前后端技术栈开发的电商商城平台，技术使用Python+FastApi框架+react + material-ui + ts+mysql，项目结构清晰、技术先进，是学习实践电商系统开发的理想案例！
### 2、源码
下载地址：https://www.3woop.com/home/Index/projectInfo?goodsId=174&typeParam=1&subKey=-1

页面有演示站，可查看预览~

开发不易，小偿提供源码，谢谢理解~

### 3、使用技术

· 编程语言（后端）：Python 3.12

· 后端框架：FastAPI

· 前端框架：React + TypeScript + Material UI

· 数据库：MySQL 8.0

· 包管理 / 构建工具：pip（后端） / npm（前端）

· 开发工具：VSCode / PyCharm
